MAKEFILE: 
1. enter "make" or "make test" will build shell.o and ./mysh, also run ./mysh automatically in an interative mode. 
2. enter "make mysh" will create a shell.o and ./mysh file only
3. enter "make valgrind" will run valgrind on ./mysh in an interative mode
4. "make clean" will remove shell.o and ./mysh
